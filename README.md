## About
Mein Ersatz für die langjährige REXX um Feiertagseinträge im Kalender zu erstellen.

## Use:
* Benötigt eine Feiertage.txt als Eingabe
* Editiere zuerst die Feiertage.txt. Du benötigst das korrekte Datum für dieses Jahr!
* Eine gute Quelle ist: https://www.ferienfeiertagedeutschland.de/2026/feiertage/
* Programm fragt nach dem zu erstellenden Jahr.    
* Schreibt eine feiertage.ics    
* Drag & Drop in den emClient Kalender. So du sie dort wieder rauslöschen willst: 
    - Kalenderdarstellung: Agenda - sortiere nach "Erstellt" ooder "Aktualisiert"
